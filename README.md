基于 STC8H8K64U 的多功能自动量程数字万用表
1. 项目简介
本项目是一款基于 STC8H8K64U 单片机开发的嵌入式数字万用表。该作品集成了电压、电阻、频率/占空比、二极管压降测量及线路通断检测功能。项目亮点在于实现了全自动量程切换逻辑，并采用了 2.5V 虚拟地技术 以支持双极性信号测量。

2. 核心功能
电压测量 (MODE_VOLT)：支持 20V 和 2000mV 双量程切换，具备零点偏移校准功能。

自动量程电阻测量 (MODE_RES)：内置四个标准基准电阻支路（200kΩ、20kΩ、2kΩ、200Ω），系统根据采样值自动升降档位。

频率与占空比测量 (MODE_FREQ)：利用单片机硬件捕获功能（INT0），可测量高达数万赫兹的频率及 0-100% 占空比。

二极管/连通性测试：支持二极管正向压降显示，并在短路（线路通断）测试时通过蜂鸣器进行声光报警。

人机交互：通过 128x64 OLED 屏幕实时显示测量数值、档位及原始 ADC 数据。

3. 硬件架构
3.1 核心组件
主控芯片：STC8H8K64U (DIP40 核心板)，主频 24MHz。

显示模块：I2C 接口 OLED 屏幕 (P2.4/P2.5)。

电源系统：USB 5V 取电，配合自锁开关控制，使用 22uF 电解电容进行低频滤波。

虚拟地电路：采用 LM358 运算放大器构建 2.5V 偏置电压，使单片机能测量负电压。

3.2 采样引脚分配
P1.3 (ADC3)：电压测量输入引脚。

P1.1 (ADC1)：电阻与二极管测量采样引脚。

P3.2 (INT0)：频率信号捕获与模式切换按键复用引脚。

P3.4 - P3.7：电阻档位切换控制引脚（控制不同功率的基准电阻）。

4. 软件逻辑说明
自动量程切换：程序实时监控 ADC 采样值。当测量值接近当前量程满量程时自动升档，当测量值过小时自动降档，确保测量精度。

按键多功能设计：短按切换子量程（如手动电阻档循环），长按切换大模式（电压 -> 电阻 -> 通断 -> 频率 -> 二极管）。

保护逻辑：在电压测量模式下，软件强制关闭所有电阻基准支路，防止高压倒灌烧毁引脚。

5. 使用与校准说明
开机：按下电源自锁开关，OLED 屏显示初始界面。

模式切换：长按按键（约 1 秒）切换不同的测量功能。

零点校准：代码中定义了 ADC_ZERO (2125)，可根据实测的 2.5V 偏置电压进行微调以消除底噪。
